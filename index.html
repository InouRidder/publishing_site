Step 1

  4. Create your file structure

    • index.html.erb

INTERMEZZO - Pictures!

We made an images folder. Lets use it.

  1. Open up the folder in your finder. Drag and drop whatever image you want to display on your website.
  2. Now we need to show them. In order to show them we need to explicitly write the path to the file. The path is always defined from the file where you want to use the picture to the picture. In the case of the index.hmtl the path to the file is "images/banner-aaa34bb2.jpg"
    Two ways of adding pictures to your site. HTML & CSS In your html 
            
          <img src="images/banner-aaa34bb2.jpg" alt="">
          <img src="images/EXACTLY THE NAME OF THE IMAGE" alt="">
  3. Or attaching it to an html element through your CSS 
            
          #banner {
            background-image: url('images/banner-aaa34bb2.jpg');
            width: 100%;
            background-size: cover;
            height: 100vh;
          }
