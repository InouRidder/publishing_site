How to deploy with github

Step 1

Sign up for Github

Step 2

Download Github Desktop Client

Step 3

Creating your local repository

  1. Create your repository
  2. Choose a name for your local repository!

Step 4

Copy your files into your repository!

  1. Find the folder through "view in finder"
  2. Open your other folder, put them next to each other and copy and paste the files

Step 5

Check, name and send it to github!

  1. Check if the files are in your repo
  2. Publish the repo through the button
  3. Choose a name!

    This name will be in your link, unless your adding a custom domain.

Step 6

Check your files on github, go to the settings.

  1. Check if your files arrived in the online repository
  2. Go to settings -> set the branch to 'master' -> press save

Step 7

Check out your website!

This can take a few minutes, github is publishing your website.

You can browse to your website using the link in the green box!
If you have bought a domain: The link in green is the link that we need to point your dns towards.
